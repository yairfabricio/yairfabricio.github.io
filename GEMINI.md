# Proyecto: Portafolio 3D "Data Nexus" - Yair Cuno Rojas

## 🌌 Alma del Proyecto
Este no es un portafolio común; es una experiencia interactiva en 3D que refleja la identidad de Yair como **Data Analyst**, pero con una estética underground y hacker inspirada en **"Mr. Robot"**. El entorno es una interfaz de terminal inmersiva y oscura (negro, verde fósforo, rojo), donde los proyectos se descubren como vulnerabilidades o bases de datos infiltradas.

## 👤 Perfil del Usuario
- **Nombre:** Yair Cuno Rojas (GitHub: yairfabricio).
- **Rol:** Data Analyst.
- **Enfoque:** Transformar datos en decisiones de negocio (Python, SQL, Storytelling).
- **Estilo deseado:** Temática "Mr. Robot" (Hacking, Oscuro, Cyber, Terminal interactiva) -> 3D Interactivo.
- **Idioma del Contenido:** El portafolio final debe estar íntegramente en **inglés**.

## 🚀 Proyectos a Incluir (The Data Nexus Nodes)

2. **First Time Response Dashboard** (`First_time_response_Power_bi`)
3. **Meta Ads Performance Dashboard** (`meta_ads_dashboard_power_bi`)

## 🛠️ Stack Tecnológico
- **Frontend:** React + Vite (TypeScript).
- **3D:** Three.js con `@react-three/fiber` y `@react-three/drei`.
- **Estilos:** Tailwind CSS.
- **Infraestructura:** GitHub CLI (`gh`) para gestión y GitHub Pages para despliegue.

## 🗺️ Mapa de Ruta (Pendiente)
1.  **Setup Limpio:** Inicializar React + Vite (usando plantilla de TypeScript de forma no interactiva).
2.  **Entorno 3D:** Crear el "Canvas" con un campo de estrellas/partículas interactivo.
3.  **Nodos de Proyectos:** Implementar los proyectos seleccionados como objetos 3D clicables.
4.  **UI Overlay:** Capa 2D interactiva en HTML/Tailwind para la descripción de los proyectos.
5.  **Despliegue:** Usar `gh` y `gh-pages`.
